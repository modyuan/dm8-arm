达梦8.4(arm64架构)镜像构建脚本

此镜像仅供测试环境使用，默认的许可证有效期很短。

## 构建镜像

本仓库仅提供达梦数据库 Docker 构建脚本，不提供任何二进制安装文件。
使用前请自行从达梦官方网站下载对应版本：
- 版本：信创版
- CPU：鲲鹏 920
- 操作系统：麒麟 V10 SP1
- 安装包文件名：dm8_20251114_HWarm920_kylin10_sp1_64.zip

解压后再把iso文件解压就可以得到`DMInstall.bin`文件。

```bash
docker build -t dm8:8.4-arm .
```

## 启动示例

见`docker-compose.yml`
